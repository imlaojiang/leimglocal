# Leimg Local - 图片本地化插件

Leimg Local 是一款用于 WordPress 的图片本地化插件，用于将文章中的外部图片下载到站点媒体库并替换为本地地址。插件支持传统编辑器（TinyMCE）与古登堡编辑器，并支持粘贴截图自动上传。

## 功能特点

1. **一键本地化外链图片**  
   在文章编辑页点击「本地化图片」，将当前内容中的外部图片自动下载到媒体库并替换链接。
2. **支持双编辑器**  
   同时兼容传统编辑器（TinyMCE）与古登堡块编辑器。
3. **粘贴截图自动上传**  
   从 QQ、微信、截图工具等复制图片后，直接粘贴进编辑器可自动上传到媒体库并插入内容。
4. **基础设置开关**  
   在后台「设置 -> 图片本地化」中可开启或关闭自动粘贴上传等选项。
5. **符合 WordPress 规范**  
   包含权限校验、Nonce 防护、国际化加载、卸载清理等基础规范。

## 环境要求

- WordPress 5.0 或更高版本
- PHP 7.4 或更高版本

## 安装方法

1. 将 `leimglocal` 文件夹上传到 `wp-content/plugins/` 目录。  
2. 登录 WordPress 后台，在「插件」中启用 **Leimg Local**。  
3. 进入「设置 -> 图片本地化」按需调整配置。  

## 使用说明

### 文章本地化

1. 新建或编辑文章。  
2. 在编辑器中点击「本地化图片」。  
3. 插件会处理当前文章中的外链图片并替换为本地地址。  
4. 点击「更新 / 发布」保存文章内容。  

### 粘贴截图上传

1. 确认「设置 -> 图片本地化」中已开启「粘贴时自动上传」。  
2. 在编辑器中直接粘贴截图图片。  
3. 图片会自动进入媒体库并插入当前位置。  

## 常见问题

### 1. 本地化会直接修改线上文章吗？
不会。点击本地化后会先修改当前编辑器内容，只有在你保存（更新/发布）文章后才会写入数据库。

### 2. 为什么有些图片本地化失败？
常见原因包括：外链地址不可访问、目标站点有防盗链、服务器网络限制、当前账户无媒体上传权限等。

### 3. 粘贴截图没有自动上传怎么办？
请检查是否开启「粘贴时自动上传」、当前用户是否有上传权限，以及浏览器是否拦截了相关脚本行为。
